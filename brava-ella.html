¡Brava Ella!

sobre el proyecto

Este proyecto fue realizado para la marca de mezcal oaxaqueña ¡Brava Ella!
Se llevó a cabo un rediseño completo de las etiquetas, generando cinco versiones distintas para cada uno de sus mezcales. Además, se desarrollaron ilustraciones personalizadas que capturan la riqueza cultural y los vibrantes colores de México, aportando así una identidad visual única a la marca.
