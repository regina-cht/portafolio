Fintenial

sobre el proyecto

El proyecto fue desarrollado para Fintenial, la comunidad que tiene de objetivo establecer una conexión con las audiencias jóvenes y prepararlas para que tomen decisiones informadas en cuanto a sus finanzas personales, proporcionando valiosos consejos de gestión financiera.

La identidad de la marca se compone de un identificador primario responsivo, la elección de un paquete de ilustraciones de código abierto, iconografía, junto con la creación de un personaje llamado "Diego", diseñado exclusivamente para representar la marca. Además, se llevó a cabo el desarrollo del feed de Instagram y se elaboró un manual normativo de marca, asegurando coherencia y consistencia en la comunicación visual de Fintenial.
